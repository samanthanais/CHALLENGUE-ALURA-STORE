# Challenge Alura Store

Este proyecto analiza los datos de ventas de las cuatro tiendas de Alura Store
con el objetivo de identificar la tienda menos eficiente y recomendar su venta.

## Tecnologías utilizadas
- Python
- Pandas
- Matplotlib

## Análisis realizado
- Ingresos totales por tienda
- Ventas por categoría
- Calificación promedio de clientes
- Productos más y menos vendidos
- Costo de envío promedio

## Conclusión
Con base en los análisis y visualizaciones, se recomienda vender la tienda con
menor rendimiento general en ingresos, satisfacción del cliente y volumen de ventas.
